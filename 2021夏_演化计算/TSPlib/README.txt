1.Individual:Initialize the citylist.
2.Population:Write information from the file to the list.
3.TSPproblem:Extract the information from the file and initialize the citylist and distance.
4.TSPtest:Extract the information from the file calculate the optimal result.
5.Crossover:The algorithm for crossover.
6.Mutation:The algorithm for mutation.
7.Selection:Thr algorithm for selection.
8.GA1:The first algorithm.
9.GA2:The second algorithm.
10.GA3:The third algoriothm.
11.Run:Extract the optimal result.
12.Client:Select parameters of population size, interation size, crossover probablity, and mutation probability.After working,it can get the optimal result.
